# TokenVault - EVM Wallet Application

## Project Overview
TokenVault is an EVM wallet application built with Tauri 2.x + React + TypeScript + TailwindCSS + Zustand.

## Current State
- Backend (Rust/Tauri 2.x): Compiles successfully with `cargo check`, 3 tests pass
- Frontend (React): Compiles successfully with `npm run build`
- Existing Tauri commands: create_wallet, import_wallet_from_mnemonic, import_wallet_from_private_key, get_wallets, export_keystore, delete_wallet
- UI Pages: WalletDashboard, CreateWallet, ImportWallet, WalletDetails (all implemented with React Router)
- Zustand store: walletStore with multi-chain support
- New RPC commands: get_supported_chains, get_balance, get_chain_info

## Already Implemented
- Wallet creation and import (mnemonic/private key)
- Wallet list and details view
- Chain selector UI (ETH, BNB, Polygon, Arbitrum, Optimism, Avalanche)
- Balance display on dashboard and details pages
- Private key export with password confirmation

## Missing Features (What Needs to be Built)
1. **Send Transaction Page** - UI form to send native tokens to another address
2. **Transaction History** - Display past transactions for a wallet (needs RPC calls)
3. **ERC20 Token Balance** - Query ERC20 token balances for a wallet address

## Technical Stack
- Frontend: React 18 + TypeScript + Vite
- Styling: TailwindCSS
- State: Zustand with persist middleware
- Routing: React Router v7
- Backend: Rust + Tauri 2.x
- HTTP Client: reqwest for RPC calls
- Build: `cargo test --manifest-path src-tauri/Cargo.toml` (Rust), `npm run build` (Frontend)

## Supported Chains (already in walletStore)
- Ethereum Mainnet (chainId: 1)
- BNB Chain (chainId: 56)
- Polygon (chainId: 137)
- Arbitrum (chainId: 42161)
- Optimism (chainId: 10)
- Avalanche (chainId: 43114)

## Acceptance Criteria
1. User can view native token balance for any supported chain
2. User can switch between chains to view different balances
3. User can send native tokens to another address
4. User can view transaction history for their wallet
5. Application builds without errors (cargo test + npm run build)
6. All new Tauri commands work correctly
